# Poker Decision Assistant (HU NLHE)

A Streamlit-based decision engine for Heads-Up No-Limit Hold'em.

## Features
- Monte Carlo equity estimation
- EV calculation for Fold / Call / Raise
- Adjustable opponent tendencies
- Session tracking & bankroll analytics

## Built With
- Python
- Streamlit
- treys (hand evaluation)
- Monte Carlo simulation
